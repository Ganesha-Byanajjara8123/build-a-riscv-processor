# build-a-riscv-processor

# 🚀 Building a 5-Stage Pipelined RISC-V Processor

## Project Overview

This repository documents my journey of designing and implementing a **5-Stage Pipelined RISC-V Processor** in **Verilog HDL**. The goal is to understand processor architecture through hands-on RTL design by implementing each hardware block individually, verifying its functionality, and integrating all modules into a complete processor.

## Learning Objectives

* Understand the RISC-V Instruction Set Architecture (ISA)
* Study processor datapath and instruction flow
* Design synthesizable RTL modules in Verilog
* Verify each module using dedicated testbenches
* Build a complete Single-Cycle Processor
* Extend the design with Hazard Detection and Data Forwarding
* Implement a 5-Stage Pipelined RISC-V Processor
