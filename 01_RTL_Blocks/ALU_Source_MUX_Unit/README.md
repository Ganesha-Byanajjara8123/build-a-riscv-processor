# ALU Source Multiplexer (ALU Src MUX)

The **ALU Source Multiplexer** selects the second operand supplied to the Arithmetic Logic Unit (ALU).

It determines whether the ALU performs operations using data from the **Register File** or an **Immediate Value** generated by the Extend Unit.

---

# Overview

Many RISC-V instructions require different ALU operands.

- Register-to-Register instructions use `rs2`
- Immediate instructions use the generated immediate

The Instruction Decoder controls this selection through the `ALU_Src_Sel` signal.

---

# Why is it Required?

| Instruction Type | Operand Selected |
|------------------|------------------|
| R-Type | Register File (`rs2`) |
| ADDI | Immediate |
| Load | Immediate |
| Store | Immediate |
| AUIPC | Immediate |
| JALR | Immediate |

Without this multiplexer, the ALU could not execute both register-based and immediate-based instructions.

---

# Features

- 2-to-1 Multiplexer
- Pure combinational logic
- Decoder-controlled selection
- 32-bit datapath
- Synthesizable Verilog HDL

---

# Interface

| Signal | Width | Description |
|---------|------:|-------------|
| `reg_data_in` | 32 | Register File output |
| `imm_data_in` | 32 | Immediate from Extend Unit |
| `ALU_Src_Sel` | 1 | Decoder control signal |
| `src2_out` | 32 | Second ALU operand |

---

# RTL Operation

```text
ALU_Src_Sel = 0

src2_out = Register File
```

```text
ALU_Src_Sel = 1

src2_out = Immediate Generator
```
