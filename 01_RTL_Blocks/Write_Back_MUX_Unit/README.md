# Write-Back Multiplexer (WB MUX)

The **Write-Back Multiplexer (WB MUX)** is the final datapath component before the Register File. It selects the correct result produced by different execution units and writes it back to the destination register.

---

# Overview

**The Blcok-Diagram of WB_MUX**

<img width="480" height="320" alt="image" src="https://github.com/user-attachments/assets/d25e5d35-01d5-4ea0-9bc1-05b52e5889cf" />

Different RISC-V instructions generate results from different hardware blocks.

Instead of connecting every block directly to the Register File, a multiplexer selects the correct source based on the control signal generated by the **Instruction Decoder**.

This module completes the **Write-Back Stage** of the processor.

---

# Why is WB MUX Required?

| Instruction Type | Write-Back Source |
|------------------|-------------------|
| R-Type / I-Type | ALU Result |
| Load Instructions | Data Memory |
| JAL / JALR | PC + 4 |
| LUI | Immediate Value |
| AUIPC | PC + Immediate |

Without this multiplexer, the processor would not know which value should be written into the destination register.

---

# Features

- Combinational RTL design
- Supports five write-back sources
- Decoder-controlled source selection
- 32-bit datapath
- Fully synthesizable Verilog HDL

---

# Interface

| Signal | Width | Description |
|---------|------:|-------------|
| `ALU_Result_in` | 32 | ALU output |
| `Loaded_data_in` | 32 | Data from Load Unit |
| `PC_plus4_in` | 32 | PC + 4 |
| `Imm_in` | 32 | Immediate value |
| `Added_data_in` | 32 | PC + Immediate (AUIPC) |
| `Result_Src_in` | 3 | Decoder select signal |
| `WB_Data_Out` | 32 | Data written back to Register File |

---

# RTL Operation

## ALU Result

```text
Result_Src = 000

WB_Data = ALU_Result
```

---

## Load Data

```text
Result_Src = 001

WB_Data = Loaded_Data
```

---

## PC + 4

```text
Result_Src = 010

WB_Data = PC_plus4
```

---

## Immediate

```text
Result_Src = 011

WB_Data = Immediate
```

---

## AUIPC

```text
Result_Src = 100

WB_Data = PC + Immediate
```
