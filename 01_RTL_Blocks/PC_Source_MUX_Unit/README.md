# PC Source Multiplexer (PC Src MUX)

The **PC Source Multiplexer** determines the next Program Counter (PC) value during instruction execution.

It selects between sequential execution (`PC + 4`) and a computed target address for branch and jump instructions.

---

# Overview

During normal execution, the processor fetches the next sequential instruction.

When a branch or jump is taken, execution must continue from a new target address.

The PC Source MUX performs this selection based on the control signal generated by the Branch Decision Logic.

---

# Why is it Required?

| PC_Sel | Next PC |
|--------:|---------|
| 0 | PC + 4 |
| 1 | Branch / Jump Target |

Without this multiplexer, the processor would always execute instructions sequentially and could never perform branches or jumps.

---

# Features

- 2-to-1 Multiplexer
- Pure combinational logic
- Branch-controlled PC selection
- 32-bit datapath
- Synthesizable Verilog HDL

---

# Interface

| Signal | Width | Description |
|---------|------:|-------------|
| `PC_plus4_in` | 32 | Sequential next address |
| `Added_data_in` | 32 | Branch/Jump target |
| `PC_Sel` | 1 | Branch control signal |
| `Next_PC_Out` | 32 | Next Program Counter |

---

# RTL Operation

```text
PC_Sel = 0

Next_PC = PC + 4
```

```text
PC_Sel = 1

Next_PC = Target Address
```
